Blazium Counter

Contador personalizable para eventos

El Blazium Counter es un addon esencial para cualquier evento o competición en Minecraft Bedrock. Ofrece un sistema de conteo personalizable que puede adaptarse a cualquier situación, desde torneos PvP hasta carreras de velocidad. Con una interfaz intuitiva y opciones de configuración avanzadas, este addon te permite mantener un seguimiento preciso de puntos, vidas, tiempo y más.

Características Principales

  • Sistema de conteo personalizable (puntos, vidas, tiempo, etc.)
  • Interfaz intuitiva y fácil de usar
  • Compatible con múltiples jugadores simultáneamente
  • Configuración avanzada para eventos complejos
  • Efectos visuales y sonoros personalizables
  • Fácil instalación y configuración

Descargar Addon

Descargar .mcaddon Descargar .zip (Manual) Ver Tutorial

Instrucciones de Instalación:

  1. Descarga el archivo .mcaddon (instalación automática) o .zip (instalación manual)
  2. Para .mcaddon: Haz clic en el archivo y sigue las instrucciones de Minecraft
  3. Para .zip: Extrae el contenido en la carpeta de behavior_packs de tu mundo
  4. Abre Minecraft Bedrock Edition y activa el addon en la configuración del mundo
  5. ¡Disfruta del addon en tu mundo!
